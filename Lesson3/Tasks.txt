- Ce face comanda: git add, git init, git commit, git push?

git add - adaugi fisierele temporar asa numit index
git init - creaza directoriu inca neexistent sau reinitializeaza unul existent
git commit - adauga modificarile in istoria proiectului
git push - incarca modificarile intr-o ramura a unui proiect.
